**Note:**

We're no longer publishing API release notes. Instead, you can find the latest updates on Shopify APIs in our [developer changelog](https://shopify.dev/changelog). You can filter updates by area. For example, you can filter API updates by the API name and version, such as GraphQL Admin API changes in version 2025-04.

Release notes list all of the changes that are introduced by an API version. Use them to check whether your app is compatible with an API version, and what changes you might need to make before you migrate your app.

---

## [Anchor to Documentation updates](/docs/api/release-notes#documentation-updates)Documentation updates

API reference documentation is updated on the first business day in Canada following the release date. For example, the API reference documentation for the 2024-01 release will be updated on January 3, 2024.
