Dynamically returns content based on the current [template](/themes/architecture/templates).

Include the `content_for_layout` object in your [layout files](/themes/architecture/layouts) between the `<body>` and
`</body>` HTML tags.

---

Note

The `content_for_layout` object is required in `theme.liquid`.
