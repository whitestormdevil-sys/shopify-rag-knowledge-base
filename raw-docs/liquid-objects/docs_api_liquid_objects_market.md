A group of one or more regions of the world that a merchant is targeting for sales.

To learn more about markets, refer to [Shopify Markets](/docs/apps/markets).
To make sure that visitors interact with the optimal version of a store using Shopify Markets,
refer to [Detect and set a visitor's optimal localization](/docs/themes/markets/localization-discovery).
