# Shopify CLI Theme commands

[theme check

Calls and runs Theme Check to analyze your theme code for errors and to ensure that it follows theme and Liquid best practices.](/docs/api/shopify-cli/theme/theme-check)[theme console

Starts the Shopify Liquid REPL (read-eval-print loop) tool.](/docs/api/shopify-cli/theme/theme-console)[theme delete

Deletes a theme from your store.](/docs/api/shopify-cli/theme/theme-delete)[theme dev

Uploads the current theme as the specified theme, or a development theme, to a store so you can preview it.](/docs/api/shopify-cli/theme/theme-dev)[theme duplicate

If you want to duplicate your local theme, you need to run `shopify theme push` first.](/docs/api/shopify-cli/theme/theme-duplicate)[theme info

Displays information about your theme environment, including your current store.](/docs/api/shopify-cli/theme/theme-info)[theme init

Clones a Git repository to your local machine to use as the starting point for building a theme.](/docs/api/shopify-cli/theme/theme-init)[theme language-server

Starts the Language Server.](/docs/api/shopify-cli/theme/theme-language-server)[theme list

Lists the themes in your store, along with their IDs and statuses.](/docs/api/shopify-cli/theme/theme-list)[theme metafields pull

Retrieves metafields from Shopify Admin.](/docs/api/shopify-cli/theme/theme-metafields-pull)[theme open

Returns links that let you preview the specified theme.](/docs/api/shopify-cli/theme/theme-open)[theme package

Packages your local theme files into a ZIP file that can be uploaded to Shopify.](/docs/api/shopify-cli/theme/theme-package)[theme profile

Profile the Shopify Liquid on a given page.](/docs/api/shopify-cli/theme/theme-profile)[theme publish

Publishes an unpublished theme from your theme library.](/docs/api/shopify-cli/theme/theme-publish)[theme pull

Retrieves theme files from Shopify.](/docs/api/shopify-cli/theme/theme-pull)[theme push

Uploads your local theme files to Shopify, overwriting the remote version if specified.](/docs/api/shopify-cli/theme/theme-push)[theme rename

Renames a theme in your store.](/docs/api/shopify-cli/theme/theme-rename)[theme share

Uploads your theme as a new, unpublished theme in your theme library.](/docs/api/shopify-cli/theme/theme-share)
